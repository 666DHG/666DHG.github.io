2022 GZEZ 冬令营游记

Posted on 2022-01-23 Edited on 2022-01-24 In Travels Valine:
Symbols count in article: 1.4k Reading time ≈ 1 mins.
Travels001

写在前面

数学集训刚回来，刚刚过去的一周确实累
不过好在冬令营取得了满意的成绩，可以稍微松一口气了
这篇文章重在记录这一届冬令营的安排，让大家了解了解
也可以给后人当成经验参考
最后附有部分笔记，可供参考

游记

Day 1

上午

16号上午，我们来到了二高（其实是回去）
9点半开营仪式，结束后就是破冰活动。
有两个游戏，第一个是班级组队踩地雷开路，挺简单的
结果第二个类似于跳珠的游戏难倒了我们大部分人
还好有其他同学聪明，我们班取得了第一

脑子是个好东西，我也好想要

接下来发了安排表，选举了班干部（还是一如既往地当工具人）

下午

下午直接开上化学，老师真猛，直接从必修一讲到选修四（反应平衡）
还好中间跳了选修三，我也提前学过一些
值得吐槽的是，老师推知识点的速度又快
然后拿几乎没水的黑笔在脏脏的白板上写板书
同时我们坐在演播厅，距离又远
真就看了个寂寞
再加上发烧，整个人生不如死，直接麻了

接着是二高劳动课程的介绍，没听

晚上

晚上的数学课，真的够水
听得十分舒服，几乎没有学什么内容
课件就放这了 【问题的提出和问题的解决】

Day 2

上午

上午物理上安培力，也做了些实验
概念理解起来不难，还讲了讲电动机与换向器

接下来就是二高特色课程，自选，我选了化工
知道了口红行业的暴利，自己做了只口红，还用盐水刻蚀

旁边兄弟还在纠结，口红是给自己妈还是给女朋友
直到老师又给每人发了一只全新包装的口红，定价 69.9 元，成本 10 元以内
还是挺好玩的

下午

下午上语文课，老师先介绍了二高的语文活动与高中语文
接下来老师介绍读书技巧，并顺带做了红楼梦导读
最后就是作文的粗略讲解，校内老师都讲过
整体上得还是很愉快的

接着是趣味运动会，考脑又考配合
我经常突发脑梗，可以说是游戏黑洞了

晚上

晚上上英语，老师教我们如何看报 + 概括新闻
也就是经典的 what-why-how 模型，大家都在摸鱼
最后，老师给我们做了篇语法填空
难得一批！10对2
全员绷不住，原来英语还可以这么玩

接着是节目排练，我们决定演个话剧

Day 3

上午

「 语文 」 ( 8:00 - 9:20 )
整体高中模式，选择题多，分值高（ 3分/道 ）
文言文很长，但有选择题帮助理解
时间紧迫，留给作文的时间不多

「 物理 」 ( 9:20 - 10:10 )
考察以初中知识为主，整体难度中等偏上
倒数第二道解答题材料多，难分析与理解
只考了一道较简单的安培力实验设计题

「 英语 」 ( 10:40 - 11:30 )
整体难度与提高卷（高中水平）相近
语法填空还好
时间较为紧迫，作文没有时间打草稿

「 化学 」 ( 11:30 - 12:00 )
整体难度适中，会涵盖初中全部知识点
有一两题考了原子半径相关内容，严重超纲
化学反应速率的大题不难

下午

「 数学 」 ( 14:20 - 15:50 )
整体难度不大，但题量大，时间极其紧迫
听老师说，这也是高考命题趋势，就是题中等难度，肯定能做出来，但没有时间磨
考察以二次函数为主，圆的相关内容不难

考完后去耕地、摘菜（二高特色经典）
我全程摸鱼，连晚上的火锅也没去
火锅的荤菜是自费的，好像大家都吃得很开心
我就默默地写话剧剧本

晚上

晚上话剧排练，累死我了
之后又修了修剧本，十点才回宿舍

Day 4

上午结营仪式

总结

其实这次冬令营还是以体验为主，没有像我们想象的那样整天上课，那么紧张
班级氛围挺好的，大家玩得来，也学得来
感觉只要好好搞好校内，上的课认真听，综合能力强一点就行

没必要紧张的，放平心态就行，大家也都很开心

笔记
